实验反思

Q4. 对于程序中定义的各个段，若段中的数据占 N 个字节，请问程序加载之后，该段实际占用的空间为？

这个问题我们在之前提到过，每个段最大长度为 64 KiB，与 16 的倍数对齐。

我们分两类情况来讨论：

  1. N % 16 == 0，那么占用大小就是 (N / 16) * 16
  2. N % 16 != 0，那么占用大小就是 (N floor 16 + 1) * 16

总结起来就是 ((N+15)/16) * 16 。

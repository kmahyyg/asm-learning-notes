3.7 转移指令(1): 操作符 OFFSET

转移指令

转移指令：可以修改 CS （或/和） IP 的指令统称为转移指令，概括的说，转移指令就是可以控制 CPU 执行内存中某处代码的指令。

转移指令的分类

指令操作 名称 示例
只修改 IP 段内转移 jmp ax
同时修改 CS 和 IP 段间转移 jmp 1000:0

段内转移又分为：

名称 修改IP的范围
短转移 -128～127
近转移 -32768～32767

转移指令分为以下几个大类：

  • 无条件转移（例如：jmp）
  • 条件转移
  • 循环（例如：loop）
  • 过程
  • 中断
